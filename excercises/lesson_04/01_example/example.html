Every Website is built out of boxes. These boxes are called div-layers.

This text is inside a div layer, which has the class "box". The class box has CSS values for margin, padding, width and background-color.
This div-layer has the same class as the div above, but has additionally the id "dark" with a darker background. The ID overwrites the value of the class.
Inside a div layer we add a new div layer with a new class.
And so on, and so on,...
Of course you can add multiplte layers on the same level. Depending on yout layout.
The three boxes in this div layer have a class that gives them a width but they are still showed below each other even if it is possible to align them next to each other.

Inside this div layer we have the same multiple div-layer with the class "left". This class gives these boxes the CSS value "float: left;" which alows the boxes to align next to each other.
When we are done using the float command we have to tell the browser that it is done. Therefore we built the class "clear" with the CSS value "clear:both;"

If we do not add this last div layer, our layout will get destroyed.
